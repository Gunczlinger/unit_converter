# Unit Converter
This app is designed to provide multiple different types of conversions at user convenience

## Conversions
- Temperature
    * Fahrenheit
    * Celcius
    * Kelvin
    * Humans (units of average human body heat)
- Distance
    * Centimeters
    * Inches
    * Feet
    * Yards
    * Meters
    * Kilometers
- Weight
    * Grams
    * Ounces
    * Pounds
    * Kilograms

## Features
- Flipping between methods of conversion
- Input sanitization
